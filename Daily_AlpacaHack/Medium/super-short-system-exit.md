`SystemExit`を発生させるためには`eval`で`exit()`を実行させればよいのですが、今回は5文字までしかペイロードを使えないのでこれだと弾かれてしまいます。

色々調べていたところ以下のページを発見しました。

[[misc] printgolf (sknbCTF 2025) 作問者writeup #Python - Qiita](https://qiita.com/claustra01/items/522f136ff8856274ea0c)

「pythonのUnicord NFKC正規化には合字（ligature）を分割するという仕様がある。例えばﬂという合字はflに変換される。そのため、ⅺとﬂを用いることで2文字減らすことができ、8文字のpayloadを作ることができる。」

よって、`exit()`の`xi`を`ⅺ`という文字に置き換えれば、最初の5文字制限を突破しつつ、`eval`で`exit()`を実行させることができるわけです。

というわけで、ペイロードは以下。

```
eⅺt()
```

文字数制限を課すjail問題はたまに見かけるので、覚えておいて損のないテクニックです。
