普通なら、以下のペイロードを送ればよい。

```json
"\"\\\"\\\\\\\"42\\\\\\\"\\\"\""
```

しかし、`if (input.includes("\\\\")) return "Invalid";`の条件に引っかかってしまうため、`\\\\`を送ることができない。

そのため、以下の様にUnicordeエスケープを使用して、`\\`を表現する必要がある。

```json
"\u0022\u005c\u0022\u005c\u005c\u005c\u0022\u005c\u005c\u005c\u005c\u005c\u005c\u005c\u002242\u005c\u005c\u005c\u005c\u005c\u005c\u005c\u0022\u005c\u005c\u005c\u0022\u005c\u0022\u0022"
```

これを送ると、FLAGを表示させることができる。
